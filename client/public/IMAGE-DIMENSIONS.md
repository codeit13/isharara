# Image & banner dimensions

Use these sizes for best quality and layout (CLS-friendly).

| Asset | Recommended size | Notes |
|-------|------------------|--------|
| **Hero banner** | **1920 × 800 px** (or 1440 × 640) | `public/images/hero-banner.png`. Used on homepage hero; aspect ~2.4:1. |
| **Bundle / offer image** | **600 × 400 px** | `public/images/bundle-gift.png`. Used in Inaugural Offer section. |
| **OG / social share** | **1080 × 1080 px** | `public/og-image.png`. Square for Facebook, Twitter, WhatsApp previews. |
| **Logo** | **512 × 512 px** (or SVG) | `public/logo.png`, `public/favicon.png`. Square; used in nav, footer, PWA. |
| **Product images** | **800 × 1067 px** (3:4) | Product cards use `aspect-[3/4]`; consistent ratio avoids layout shift. |

All dimensions are in pixels. Export as PNG or WebP; keep file sizes reasonable for fast loading.
